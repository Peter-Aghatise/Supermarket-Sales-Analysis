# Supermarket Sales Analysis

## Project Overview
Analysis of supermarket sales data across three cities using Python, Pandas, and Matplotlib.

## Business Questions Answered
- Which city generates the most sales?
- What is the gender distribution of customers?
- Which payment method is most popular?
- What is the average customer rating?
- Which product line performs best?

## Tools Used
- Python
- Pandas
- Matplotlib
- MySQL

## Key Findings
- Naypyitaw generates the highest average income per transaction
- Female customers account for 59% of transactions
- Ewallet is the most popular payment method
- Food and beverages received the highest customer rating

## Files
- supermarket_analysis.py — Exploratory data analysis
- supermarket_cleaning.py — Data cleaning pipeline
- visualization.py — Dashboard and charts
